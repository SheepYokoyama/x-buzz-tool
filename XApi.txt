X API 設定手順
1. X Developer Portal でアプリを作成
https://developer.twitter.com/en/portal/dashboard にアクセス
「+ Add App」でアプリを作成
User authentication settings で以下を設定：
App permissions → Read and Write
Type of App → Web App
Callback URL → http://localhost:3000（開発時）
2. 認証情報を取得
「Keys and Tokens」タブから以下を取得：

環境変数	取得場所
X_API_KEY	Consumer Keys → API Key
X_API_SECRET	Consumer Keys → API Key Secret
X_ACCESS_TOKEN	Authentication Tokens → Access Token
X_ACCESS_TOKEN_SECRET	Authentication Tokens → Access Token Secret
X_BEARER_TOKEN	Authentication Tokens → Bearer Token
3. .env.local に値を設定
X_API_KEY=取得した値
X_API_SECRET=取得した値
X_ACCESS_TOKEN=取得した値
X_ACCESS_TOKEN_SECRET=取得した値
X_BEARER_TOKEN=取得した値

追加したAPIルート
エンドポイント	機能	必要なプラン
POST /api/x/tweet	Xに投稿	Free
GET /api/x/metrics/[tweetId]	インプレッション取得	Free（public_metrics） / Basic以上（organic_metrics）
注意：インプレッション数（impression_count）の取得は X API の Basic プラン（$100/月）以上が必要です。Free プランではいいね・RT数のみ取得可能です。
